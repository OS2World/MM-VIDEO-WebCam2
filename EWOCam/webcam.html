Welcome to my Web Camera.
A WebCamera is a video camera, that takes pictures in intervals and shows them in the web.
My WebCamera is placed inside my office, looking through the window down to the Elbe river.
The Elbe is one of Germanys biggest rivers and connects Hamburg to the North Sea.
My office is in Geesthacht at the GKSS Research Center, approx. 30km east of Hamburg.

Hard- and Software I use

  • Intel Pentium(R)/133MHz with 64MB RAM
  • ELSA WINNER 2000/Office Graphics Adapter with Video In/Out
  • A monochrome Stemmer CCD camera (courtesy GKSS)
  • IBM OS/2 Warp Version 4.0 with Internet Connection Server
  • VideoIn driver from ELSA Download page
  • A program "EWOCam/2" (ELSA WINNER OFFICE Cam for OS/2), written by myself, adapted from my WebCam/2 app for the FAST Movie Machine

What is EWOCam/2?

Click here for the live camera

EWOCam/2 takes timed camera snapshots and saves them as images to disk.
Primary I wrote this app for myself, but it came out that many OS/2 users want to have this, too...

EWOCam/2's current developement status:

The standard image format is OS/2 bitmap (.BMP). Resolution is 160 x 120, 24 bit color, should be possible to enhance it. With a user exit file ("WCAMEXIT.CMD") it is possible to start a conversion program or manipulate the bitmap after grabbing a new one. Native JPEG support is planned for the future.
